# AGENTS.md

## Project

TypeScript CLI for composing and exporting Path of Exile item filters. Custom filters live in `src/filters/`; shared filter sections and defaults live in `src/filters/shared/`.

## Working conventions

- Run `npm run check` before completing TypeScript changes.
- Use `npm run export <filter-name>` to regenerate types and sounds, sync sounds, and export one filter.
- `src/types/autogenerated/` and `src/types/sounds/generated-sounds.ts` are generated; change their source data or generation scripts rather than editing them directly.
- Personal filter folders and `src/filters/shared/user-defaults.ts` / `user-styles.ts` are intentionally gitignored. Do not add them to commits.
- Read `README.md` for filter creation, export setup, and sound configuration.
